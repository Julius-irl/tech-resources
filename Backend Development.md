**📌 Backend Development Resources**

**🆓 Free Resources**

🔗 **roadmap.sh — Backend Developer Roadmap**  
[link](https://roadmap.sh/backend) — A good starting point if you don't know where to begin, or what order to learn things in.

🔗 **freeCodeCamp — Back End Development and APIs**
[link] — They offer a free, project based course that covers Node.js and Express, building REST APIs, and connecting to databases. Also comes with the chance to earn a certification at the end.

🔗 **MDN Web Docs — Server-Side Programming**  
[link](https://developer.mozilla.org/en-US/docs/Learn_web_development/Extensions/Server-side) — Solid for learning backend fundamentals. Goes over request/response cycles, and server side concepts.

🔗 **PostgreSQL — Official Documentation**  
[link](https://www.postgresql.org) — The official doc for PostgreSQL, one of the most widely used SQL databases. A good starting point if you want to understand structured data, and relationships between tables.

🔗 **MongoDB — Official Documentation**  
[link](https://www.mongodb.com) — The official docs for MongoDB, the most common NoSQL document database. Good starting point if you're working with more flexible, unstructured data or want to understand the alternative to relational databases.
